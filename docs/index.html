    📒 USPG Assistant - v0.1.0

    USPG Assistant es una aplicación web diseñada para ayudarte a organizar tu día de manera sencilla y eficiente. Está construida con tecnologías modernas como Next.js y TypeScript, y sigue los principios de la Arquitectura Limpia (Clean Architecture) para mantener una estructura sólida y escalable.

    🚀 Propósito

    La aplicación permite:

    • ✅ Gestionar notas y tareas.
    • ⏰ Asignar fechas de vencimiento.
    • 📅 Visualizar tareas en un calendario interactivo.
    • 🔔 Recibir notificaciones de vencimientos.

    🛠️ Tecnologías utilizadas

    • Next.js – Framework para React con renderizado híbrido.
    • TypeScript – Tipado estático para mayor robustez.
    • Clean Architecture – Separación entre dominio, aplicación, interfaz e infraestructura.
    • Firebase – Base de datos NoSQL (Firestore) y autenticación.
    • TypeDoc – Generación automática de documentación.

    🔐 Autenticación

    • Inicio de sesión con Google.
    • Inicio de sesión con correo y contraseña tradicional.

    📋 Funcionalidades principales

    To-Do List

    • Bandeja de notificaciones (próximas a vencer o vencidas).
    • Aqui el usuario puede agregar una tarea con los campos requeridos para luego organizar su día a traves del calendario

    📝 Notas

    • Crear, editar, eliminar y buscar notas personales.
    • Organización por reciente y antiguo.

    📅 Calendario

    • El usario puede ver una vista mensual de tareas con acceso a detalles.
    • Puede ver el estado visual de tareas (pendiente/completada).
